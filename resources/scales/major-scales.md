# メジャースケール一覧

## 構造

全-全-半-全-全-全-半（W-W-H-W-W-W-H）

---

## 12キーのメジャースケール

### Cメジャー（フラット/シャープなし）
```
C - D - E - F - G - A - B - C

指板（3弦3フレット起点）：
|1 e + a |2 e + a |3 e + a |4 e + a |
|C D E F |G A B C |. . . . |. . . . |
```

### Gメジャー（♯1つ：F#）
```
G - A - B - C - D - E - F# - G
```

### Dメジャー（♯2つ：F#, C#）
```
D - E - F# - G - A - B - C# - D
```

### Aメジャー（♯3つ：F#, C#, G#）
```
A - B - C# - D - E - F# - G# - A
```

### Eメジャー（♯4つ：F#, C#, G#, D#）
```
E - F# - G# - A - B - C# - D# - E
```

### Bメジャー（♯5つ：F#, C#, G#, D#, A#）
```
B - C# - D# - E - F# - G# - A# - B
```

---

### Fメジャー（♭1つ：Bb）
```
F - G - A - Bb - C - D - E - F
```

### Bbメジャー（♭2つ：Bb, Eb）
```
Bb - C - D - Eb - F - G - A - Bb
```

### Ebメジャー（♭3つ：Bb, Eb, Ab）
```
Eb - F - G - Ab - Bb - C - D - Eb
```

### Abメジャー（♭4つ：Bb, Eb, Ab, Db）
```
Ab - Bb - C - Db - Eb - F - G - Ab
```

### Dbメジャー（♭5つ：Bb, Eb, Ab, Db, Gb）
```
Db - Eb - F - Gb - Ab - Bb - C - Db
```

### Gbメジャー（♭6つ：Bb, Eb, Ab, Db, Gb, Cb）
```
Gb - Ab - Bb - Cb - Db - Eb - F - Gb
```

---

## 五度圏での覚え方

```
        C
    F       G
  Bb          D
 Eb            A
  Ab          E
    Db      B
       Gb/F#

時計回り：♯が増える（G→D→A→E→B）
反時計回り：♭が増える（F→Bb→Eb→Ab→Db）
```

## 指板上のポジション

### ポジション1（ルートが4弦）

```
4弦: R - 2 - 3
3弦:         4 - 5 - 6
2弦:             7 - R

例：Cメジャー（3フレット起点）
E|---------------|
A|---------------|
D|------5-7------|
G|--3-5------8-10|
```

### ポジション2（ルートが3弦）

```
3弦: R - 2 - 3
2弦:         4 - 5 - 6
1弦:             7 - R

例：Cメジャー（3弦3フレット起点）
E|---------------|
A|------5-7------|
D|--3-5------7-8-|
G|---------------|
```

## 練習方法

1. **1キーずつ**
   - 今週はCメジャー、来週はGメジャー...

2. **上行→下行**
   - ルートから始めてルートに戻る

3. **メトロノームで**
   - BPM 60から始めて徐々に上げる

4. **様々なポジションで**
   - 同じスケールを違う位置で弾く
