# ベース指板リファレンス（4弦・標準チューニング）

## 開放弦
- 4弦 = E1
- 3弦 = A1
- 2弦 = D2
- 1弦 = G2

## 全フレット音名表（0F〜12F）

```
     0F    1F    2F    3F    4F    5F    6F    7F    8F    9F   10F   11F   12F
    ─────────────────────────────────────────────────────────────────────────────
1弦 │ G  │ G#/Ab│  A  │ Bb  │  B  │  C  │ C#/Db│  D  │ Eb  │  E  │  F  │ F#/Gb│  G  │
    ─────────────────────────────────────────────────────────────────────────────
2弦 │ D  │ Eb  │  E  │  F  │ F#/Gb│  G  │ G#/Ab│  A  │ Bb  │  B  │  C  │ C#/Db│  D  │
    ─────────────────────────────────────────────────────────────────────────────
3弦 │ A  │ Bb  │  B  │  C  │ C#/Db│  D  │ Eb  │  E  │  F  │ F#/Gb│  G  │ G#/Ab│  A  │
    ─────────────────────────────────────────────────────────────────────────────
4弦 │ E  │  F  │ F#/Gb│  G  │ G#/Ab│  A  │ Bb  │  B  │  C  │ C#/Db│  D  │ Eb  │  E  │
    ─────────────────────────────────────────────────────────────────────────────
```

## よく使う音の位置

### C の位置
- 3弦 3F
- 2弦 10F
- 1弦 5F
- 4弦 8F

### G の位置
- 1弦 0F（開放）
- 2弦 5F
- 3弦 10F
- 4弦 3F

### ナチュラルハーモニクス
- 5F：開放弦の2オクターブ上
- 7F：開放弦の1オクターブ+5度上
- 12F：開放弦の1オクターブ上

## 半音・全音の確認

- 半音 = 1フレット移動
- 全音 = 2フレット移動

### 自然に半音の関係にある音
- E - F（半音）
- B - C（半音）

### それ以外は全音
- C - D（全音）
- D - E（全音）
- F - G（全音）
- G - A（全音）
- A - B（全音）
