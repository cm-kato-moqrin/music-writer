# 理論学習サポート

音楽理論をベーシスト視点で解説します。

## 引数

$ARGUMENTS - 学習したいトピック

例：
- `/theory ダイアトニックコード`
- `/theory ドリアンモード`
- `/theory II-V-I`
- `/theory テンション`

## 出力形式

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📚 {トピック名}
━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 概念の説明
{分かりやすい説明}

■ ベースでの実践
{指板上での位置、弾き方}

ポジション例（Key=C の場合）：
{TABや指板図をテキストで}

■ よく使う場面
- {場面1}
- {場面2}

■ 練習フレーズ
{具体的なフレーズをテキストで}
例：C-E-G-B（Cmaj7アルペジオ）

■ 関連する曲
- {曲名} - {どこで使われているか}

■ 確認クイズ
Q: {理解度確認の質問}

━━━━━━━━━━━━━━━━━━━━━━━━━━━
💡 関連トピック：{関連するトピック}
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 対応トピック例

### スケール系
- メジャースケール
- ナチュラルマイナースケール
- ペンタトニックスケール
- ブルーススケール
- モード（ドリアン、ミクソリディアン、リディアン等）
- オルタードスケール

### コード系
- ダイアトニックコード
- セカンダリードミナント
- 代理コード
- テンションコード
- ディミニッシュコード

### 進行系
- II-V-I
- I-V-VI-IV
- 12小節ブルース
- ターンアラウンド
- モーダルインターチェンジ

### その他
- 五度圏
- インターバル（音程）
- コードトーン
- アプローチノート

## 注意

- 常にベーシスト視点で説明する
- 理論だけでなく「指板上でどう弾くか」を必ず含める
- 難しい用語は噛み砕いて説明する
- 実際の曲での使用例を挙げる
- クイズで理解度を確認する機会を提供する
