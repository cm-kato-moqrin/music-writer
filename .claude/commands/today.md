# 今日の練習メニュー生成

今日の練習メニューを生成してください。

## 手順

1. まず `plans/master-plan.md` を読んで開始日を確認
2. 開始日から今日が何週目かを計算
3. 該当フェーズの `plans/phase{N}/weekly-curriculum.md` を参照
4. 今日の曜日に応じたメニューを生成
5. `progress/weekly-log.md` から前回の引き継ぎ事項を確認

## 出力形式

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📅 {日付}（{曜日}）
📍 第{N}期・第{M}週目
⏱️ 目安：{60分 or 90分}
━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ ウォームアップ（10分）
├── {内容}
└── {内容}

■ 技術練習（15分）
├── テーマ：{その週のテーマ}
├── 意識ポイント：{具体的なポイント}
└── 参考曲：{あれば}

■ 理論学習（10-15分）
├── トピック：{その週のトピック}
├── 確認：{具体的な内容}
└── 練習：{指板上での確認方法}

■ ドラム連携（10-15分）
├── パターン：{その週のパターン}
├── 練習：{具体的な練習内容}
└── 意識：{ポイント}

■ 曲練習（15-20分）
├── 課題曲：{曲名}
└── 目標：{今日の目標}

■ ジャコ要素（金曜/週末のみ）
├── テーマ：{その週のジャコ要素}
└── 練習：{具体的な練習}

■ 作曲・編曲（週末のみ・15分）
├── 課題：{内容}
└── 目標：{今日の目標}

━━━━━━━━━━━━━━━━━━━━━━━━━━━
💡 前回からの引き継ぎ：
{あれば表示}
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 曜日別の重点

- 月：技術 + 理論
- 火：ドラム連携 + 曲練習
- 水：技術 + 理論
- 木：ドラム連携 + 曲練習
- 金：技術 + ジャコ要素
- 土：総合 + 作曲・編曲（90分）
- 日：振り返り / 自由練習（90分 or 休み）

## 注意

- 開始日が未設定の場合は、ユーザーに設定を促す
- 練習時間は平日60分、週末90分を基本とする
- ユーザーの進捗に応じて柔軟に調整を提案する
