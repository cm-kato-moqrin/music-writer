# ジャコ・パストリアス スタイル学習

ジャコ・パストリアスの特徴的な技法とフレージングを学びます。

## 引数

$ARGUMENTS - 学習したいテーマ

例：
- `/jaco ハーモニクス`
- `/jaco ドリアンフレーズ`
- `/jaco 16分ライン`
- `/jaco ダブルストップ`
- `/jaco Continuum`
- `/jaco Teen Town`
- `/jaco レビュー`

## ジャコの主要な特徴（参照用）

| 要素 | 説明 |
|-----|------|
| ハーモニクス | 人工・ナチュラル両方を多用 |
| 16分フレーズ | 滑らかで歌うようなライン |
| ドリアン | マイナー系でよく使用 |
| クロマチック | 半音アプローチを多用 |
| ダブルストップ | 10度などの2音同時 |
| モード横断 | 1フレーズで複数モードを通過 |
| メロディック | ベースでメロディを歌う |

## 注意

- ジャコのスタイルは高度なので、ユーザーのフェーズを考慮する
- 第1-2期は基礎要素（ハーモニクス、アルペジオ）中心
- 第3-4期で本格的なフレーズ作成へ
- 技術だけでなく「歌心」の重要性も伝える
