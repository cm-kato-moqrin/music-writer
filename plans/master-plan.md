# 半年マスタープラン：リズムセクション総合力向上

## 目標（6ヶ月後）

1. **演奏力**：ジャズ、ポップス、ロック、**特にファンク**など様々なジャンルの演奏ができる
2. **編曲・作曲力**：楽譜（グリッド可）を用いて、音楽理論を活用しながら編曲・作曲ができる
3. **読譜・採譜力**：楽譜を読んで演奏できる、聴いた音楽を楽譜に起こせる

### 演奏イメージ
- **ベース**：ジャコ・パストリアス風のメロディックで技巧的なフレーズ
- **ギター**：Larry LaLonde（Primus）風の実験的でファンキーなリフ（アドリブ不要）

---

## 現状評価

| 項目 | ベース | ギター |
|-----|--------|--------|
| 技術 | Fleaコピー可 | BSSMコピー可（正確性は劣る） |
| 理論知識 | ダイアトニック、機能和声、代理コード | 同左 |
| 指板記憶 | 不十分 | 不十分 |
| 知識の実践化 | 未統合 | 未統合 |
| 五線譜→即演奏 | 読めるが直結しない | 同左 |
| リズム譜 | 複雑なものは難しい | 同左 |

**結論**：技術・知識は十分。「指板記憶」と「知識の統合」に集中すれば半年で達成可能。

## 3期構成（各2ヶ月）

### 第1期（1-2月目）：基礎統合
**テーマ**：知識と指板を繋げる

| 週 | ベース（月水金） | ギター（火木土） |
|---|-----------------|-----------------|
| 1-2 | 指板音名記憶（ナチュラル音） | 指板音名記憶（ナチュラル音） |
| 3-4 | 指板音名記憶（♯/♭含む全音） | 指板音名記憶（♯/♭含む全音） |
| 5-6 | 度数で弾く訓練（I-IV-V-I等） | コードトーンの指板把握 |
| 7-8 | モード実践（ドリアン、ミクソリディアン） | テンションボイシング基礎 |

**マイルストーン**
- [ ] 任意の音を全弦で2秒以内に押さえられる
- [ ] 任意のキーでI-IV-V-Iを即座に弾ける
- [ ] ドリアン、ミクソリディアンを任意のキーで弾ける
- [ ] 基本コードにテンションを加えられる（ギター）
