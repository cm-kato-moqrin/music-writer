# 1年間マスタープラン：ベーシストとしての総合力向上

## 目標
1年後に、ジャズ・ポップ・ロック・ファンクなど多様なジャンルで演奏・編曲・作曲ができるベーシストになる。特にジャコ・パストリアスのようなメロディックで技巧的なフレーズを作れるようになる。

## 現状
- 演奏歴：1-3年
- 技術：ピチカート、スラップ、タッピング、和音など幅広い奏法を習得済み
- 理論：スケール・コード知識はほぼなし（耳コピで対応）
- 強み：耳コピ能力がある

## 練習時間
- 平日：60分
- 週末：90分

---

## 4期構成

### 第1期（1-3月目）：基礎固め
**テーマ**：理論の土台を作り、基本グルーヴを安定させる

| 分野 | 内容 |
|-----|------|
| 演奏 | 8ビート、16ビート、シャッフルの基礎グルーヴ |
| 理論 | メジャー/マイナースケール、ダイアトニックコード |
| ドラム連携 | 基本ビート認識、キックとのユニゾン |
| 作曲・編曲 | コード進行上のベースライン作成 |
| ドラム作成 | 基本8ビート、16ビートパターン |
| ジャコ要素 | ナチュラルハーモニクスの位置把握、基礎アルペジオ |

**マイルストーン**
- [ ] 12キーのメジャースケールを指板上で弾ける
- [ ] ダイアトニックコード（I〜VII）を理解している
- [ ] 8ビートでドラムトラックと安定して合わせられる
- [ ] 簡単なコード進行にベースラインをつけられる

---

### 第2期（4-6月目）：応用展開
**テーマ**：モードを理解し、より動きのあるベースラインを弾く

| 分野 | 内容 |
|-----|------|
| 演奏 | ウォーキングベース、ゴーストノート、16分フレーズ |
| 理論 | モード（ドリアン、ミクソリディアン）、II-V-I |
| ドラム連携 | ファンク、ボサノバ、スウィングパターン |
| 作曲・編曲 | リハーモナイズ、リフ作成 |
| ドラム作成 | ファンク/R&Bパターン |
| ジャコ要素 | 人工ハーモニクス、クロマチックアプローチ、ドリアンフレーズ |

**マイルストーン**
- [ ] ドリアン、ミクソリディアンを実践で使える
- [ ] II-V-I進行でウォーキングベースが弾ける
- [ ] ファンクの16分グルーヴが安定している
- [ ] 人工ハーモニクスの基本ができる

---

### 第3期（7-9月目）：ジャンル横断 + ジャコ深掘り
**テーマ**：複数ジャンルを行き来し、ジャコ的フレージングを習得

| 分野 | 内容 |
|-----|------|
| 演奏 | ジャズスタンダード、ファンク実践 |
| 理論 | テンションコード、代理コード、リディアン/オルタード |
| ドラム連携 | フィル対応（維持/呼応/休符）、インタープレイ基礎 |
| 作曲・編曲 | オリジナルベースライン、ジャンル変換 |
| ドラム作成 | ジャズスウィング、フィルパターン |
| ジャコ要素 | ダブルストップ、16分音符の流れるライン、モード横断 |

**ジャコ課題曲分析**
- Come On Come Over
- Continuum
- Teen Town

**マイルストーン**
- [ ] ジャズスタンダードでウォーキング+ソロができる
- [ ] テンションコードを理解し、フレーズに活かせる
- [ ] フィル中のベース対応を3パターン使い分けられる
- [ ] ジャコ風の16分フレーズを自分で作れる

---

### 第4期（10-12月目）：統合・創作
**テーマ**：学んだ全てを統合し、オリジナル作品を作る

| 分野 | 内容 |
|-----|------|
| 演奏 | 即興演奏、セッション対応、メロディック・ベース |
| 理論 | 総合的なハーモニー、アナライズ能力 |
| ドラム連携 | 即興対応、ポリリズム |
| 作曲・編曲 | オリジナル楽曲制作 |
| ドラム作成 | オリジナルパターン、変拍子 |
| ジャコ要素 | ジャコ・スタイルのオリジナルフレーズ/ソロ作成 |

**マイルストーン**
- [ ] 初見のコード進行で即興ができる
- [ ] ドラマーとインタープレイができる
- [ ] オリジナル楽曲を1曲以上完成させる
- [ ] ジャコ・スタイルのソロを作曲できる
